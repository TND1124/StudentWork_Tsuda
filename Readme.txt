【作品詳細】
フォルダ 2つ
GameLibrary	DirectX10を使用した自作のゲームライブラリです。
		自作のフォーマットのnxファイルを読み込んで、
		スキンメッシュアニメーションを行えます。

操作方法
Z                ・・・ 攻撃
矢印キー         ・・・ 移動
矢印キー + Shift ・・・ 走る
XまたはC         ・・・ カメラ回転 

XConvertNX	xファイルから自作フォーマットのnxファイルに
		変換するプロジェクトです。
		変換したファイルはGameLibraryで使用できます。


【一番良い作品】
